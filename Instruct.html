Instructions:

This is if you want to add files to YourSurveys Note: this only for people who wanted a request of a survey.



  1. Download the file/Link it.
  2. Send the file into the folder this is in.
  3. Then download Notepad++ to edit 'YourSurveys'
  4. Open Notepad++
  5. Then open 'YourSurvey'
  6. Go to 'ul' tag and click between them.
  7. Then type "li[Make sure you add <...>, replace ... to li]" and "li[Copy the ... tag but add a / to it and put /li into it.]"
  8. Then type in between the li tags "" and ""[Turn - into one word]
  9. In the a tag between replace - to {space} href="filename"
  10. Now in between type >"{Name of survey}, 'Latest updated:' {Current date(by finished updated)}, 'Made by:{Your Username/Name}'"<
  11. if add link type The same format as above then type ,'EnableLink:SurveyMaker.com/links/enable/[page name]/id=[page ID(get from owner)]'
